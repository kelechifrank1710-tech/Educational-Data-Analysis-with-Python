# Educational-Data-Analysis-with-Python
Python-based educational data analysis exploring student outcomes through data cleaning, EDA, visualization, and statistical analysis.
This repository documents my learning journey in educational data analysis using Python. It includes data cleaning, preprocessing, exploratory data analysis (EDA), visualization, and statistical analysis of educational datasets.
## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
